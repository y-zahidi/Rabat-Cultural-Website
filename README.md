## At a glance

Rabat Cultural is a digital storytelling platform for Morocco's capital — a bridge between the city's UNESCO heritage and its contemporary identity, built so that *anyone* can use it (great accessibility, fast on slow connections, beautiful on all screens).

- **8 thematic sections** — shopping, sports, beaches, nature, historic medina, transport, neighborhoods, practical info
- **80,000+ words** of original, researched content
- **WCAG 2.1 AA compliant** — keyboard-friendly, screen-reader friendly
- **92+ Mobile / 98+ Desktop** on Google PageSpeed
- **Modern WordPress** — Block Theme + Full Site Editing (FSE) + 50+ custom block patterns

> 🌐 **See it live:** <https://rabatculture.lovestoblog.com>

## Why I built this

Rabat is a city with layers — Roman ruins, an Andalusian medina, modernist civic architecture, beaches, modern tramways. Most digital coverage either freezes it as a tourist postcard or buries the good stuff under TripAdvisor noise. I wanted to build a single platform that:

1. **Tells the story properly** — text written, not scraped
2. **Loads fast on a 3G phone** in the medina
3. **Works for everyone** — including users on screen readers and keyboards
4. **Lives in a system non-technical contributors can edit** — hence WordPress

## What I learned building this

- **WordPress Block Themes are a real upgrade.** Coming from classic themes, the FSE approach + `theme.json` tokens cut my CSS to a fraction.
- **Performance starts at the asset level.** The biggest PageSpeed wins came from converting hero images to WebP and serving correctly-sized variants — not from any clever JS trick.
- **Accessibility is a discipline, not a checkbox.** Every component had to be tested with the keyboard *and* with VoiceOver / NVDA. The WCAG 2.1 AA score didn't come from a plugin.
- **Content is the hard part.** 80,000 words researched and written takes longer than the entire codebase.

## Design language

A cohesive palette inspired by Moroccan aesthetics — terracotta and ochre against deep blues, with Arabic-friendly typography and ornamental flourishes used sparingly. Every color is documented in `theme.json` so it can be reused across patterns.
